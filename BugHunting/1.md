# 🐛 Setup Bug Bounty / Hunting Environment

Panduan ini dibuat untuk menyiapkan environment **Bug Bounty / Web Security Testing** dari nol hingga siap dipakai.

## 1. Persiapan Dasar

### Tujuan

* Web Hacking (Bug Bounty)
* Reconnaissance
* Automation (Python / scripting)


## 2. Instalasi OS

### Opsi 1 (Disarankan)

Gunakan **Kali Linux**

#### Download

* [https://www.kali.org/get-kali/](https://www.kali.org/get-kali/)

#### Install

* Via **Virtual Machine** (VirtualBox / VMware) 👉 paling aman
* Atau **Dual Boot** (lebih performa, tapi berisiko kalau salah setup)


### Opsi 2 (Advanced User)

* Linux biasa (Void / Arch / Ubuntu)
* Install tools manual (lebih ringan & fleksibel)

## 3. Update Sistem

```bash
sudo apt update && sudo apt upgrade -y
```

## 4. Tools Wajib Bug Bounty

### Recon & Enumeration

* **subfinder** → cari subdomain
* **amass** → enumerasi lengkap
* **assetfinder**

### Scanning

* **nmap** → scan port & service
* **httpx**

### Web Testing

* **Burp Suite** → intercept & testing
* **ffuf** → directory brute force

### URL & Parameter

* **waybackurls**
* **gau**

### Automation

* **Python** (untuk scripting)
* Bash scripting

## 5. Install Tools (Cepat)

```bash
sudo apt install -y git curl wget nmap python3 python3-pip
```

### Install Go (untuk tools modern)

```bash
sudo apt install golang -y
```

Tambahkan PATH `.bashrc` atau `.zshrc`:

```bash
export PATH=$PATH:$(go env GOPATH)/bin
```

### Install Tools Go

```bash
go install github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
go install github.com/projectdiscovery/httpx/cmd/httpx@latest
go install github.com/ffuf/ffuf@latest
go install github.com/tomnomnom/assetfinder@latest
go install github.com/lc/gau/v2/cmd/gau@latest
```

## 6. Setup Browser

Gunakan:

* Firefox (default Kali)

Tambahkan:

* FoxyProxy
* Wappalyzer

## 7. Setup Burp Suite

1. Jalankan **Burp Suite**
2. Set proxy:

   ```
   127.0.0.1:8080
   ```
3. Import certificate ke browser


## 8. Update Tools

```bash
sudo apt update
go install ...@latest
```

## 9. Etika & Legal

* Hanya tes target yang **diizinkan**
* Gunakan platform:

  * HackerOne
  * Bugcrowd

## 10. Troubleshooting

### Tools tidak ditemukan

```bash
echo $PATH
```

### Go tools tidak jalan

```bash
export PATH=$PATH:$(go env GOPATH)/bin
```

### Burp tidak intercept

* cek proxy
* cek certificate

## 💡 Tips

* Fokus ke **1 target dulu**
* Jangan install semua tools → pahami dulu
* Automation > manual (kalau sudah paham)
