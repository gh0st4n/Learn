# Setup Bug Bounty / Hunting Environment

Panduan ini dibuat untuk menyiapkan environment **Bug Bounty / Web Security Testing** dari nol hingga siap dipakai.

## 1. Persiapan Dasar

### Tujuan

* Web Hacking (Bug Bounty)
* Reconnaissance
* Automation (Python / scripting)


## 2. Instalasi OS

### Opsi 1 (Disarankan)

Gunakan **Kali Linux**

#### Download

* [https://www.kali.org/get-kali/](https://www.kali.org/get-kali/)

#### Install

* Via **Virtual Machine** (VirtualBox / VMware) 👉 paling aman
* Atau **Dual Boot** (lebih performa, tapi berisiko kalau salah setup)


### Opsi 2 (Advanced User)

* Linux biasa (Void / Arch / Ubuntu)
* Install tools manual (lebih ringan & fleksibel)

## 3. Update Sistem

```bash
sudo apt update && sudo apt upgrade -y
```

## 4. Etika & Legal

* Hanya tes target yang **diizinkan**
* Gunakan platform:

  * HackerOne
  * Bugcrowd

## 💡 Tips

* Fokus ke **1 target dulu**
* Jangan install semua tools → pahami dulu
* Automation > manual (kalau sudah paham)
