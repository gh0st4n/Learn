# Variabel

Variabel adalah referensi dari sebuah value. Variabel sering di analogikan seperti sebuah wadah yang menampung sebuah benda.

Python memiliki beberapa peraturan dalam penulisan sebuah variabel. Jika kamu ingin melihat detail aturan tersebut bisa kunjungi laman [PEP8](https://peps.python.org/pep-0008/#type-variable-names) ini.

Peraturan tersebut adalah:
1. Menggunakan Kata tanpa didahului dengan angka
```python
# Benar
nama = "bob"

# Salah
1nama = "bob"
```
2. Menggunakan underscore, jika ingin menggunakan kata panjang
```python
# Benar
angka_saya = 12

# Salah
12angka_saya = 12
```
3. Variabel Python sangat sensitif pada penggunan huruf Besar/Keci
```python
Nama_saya = "james"
nama_saya = "joko"
print(Nama_saya)
print(nama_saya)

# outputnya :
# james
# joko
```
**Pada Python, penggunaan variabel disarankan menggunakan `snake_case`.**
